### To watch for the changes within Storybook components, use the following command

```bash
lando ssh --command "watch --color drush storybook:generate-all-stories"
```

### To generate the Storybook components once use the following command

```bash
lando drush storybook:generate-all-stories
```

### Running development environment && Storybook

```bash
lando npm run develop
```
